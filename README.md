# Amazon Landing Page Clone

A pixel-perfect, fully responsive clone of Amazon's
landing page built with pure HTML5 and CSS3.

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
- CSS Animations

---

## 🚀 Live Demo

[ Click here to view live site](https://amazondesktop.netlify.app)

---

## ✨ Features

- Fully responsive across mobile, tablet and desktop
- Pixel-perfect Amazon UI recreation
- Multi-section responsive layout
- Product grid using CSS Grid
- Responsive navigation bar with search bar
- Hero section with banner
- Product categories section
- Footer with multiple columns
- Hover transitions and animations
- Cross-browser compatible
- Clean and well-structured code
