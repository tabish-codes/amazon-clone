# Amazon Landing Page Clone

A responsive clone of Amazon's landing page
built with pure HTML5 and CSS3.

## Built With
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design

## Live Demo
[View Live Site](https://amazondesktop.netlify.app)

## Features
- Multi-section responsive layout
- Product grid using CSS Grid
- Responsive navigation bar
- Hover transitions and animations
- Cross-browser compatible
